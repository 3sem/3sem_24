## Usage
To build the project go to [UsoltsevI/task2/](UsoltsevI/task2/) folder and run the following command:
```
cmake -DCMAKE_BUILD_TYPE=Release -S . -B build
```

Then use it:
```
VERBOSE=1 cmake --build ./build
```

Finally, to run the program, tap:
```
./build/bigpipe
```
